0x01 解

拿到是一个exe

查壳，发现是pyinstaller

PyInstaller Extractor is a Python script to extract the contents of a PyInstaller generated executable file.

然后使用网站进行反编译：https://www.lddgo.net/string/pyc-compile-decompile

当然也可以直接使用pycdc

不过本地运行输出的结果内的中文部分是乱码

这题似乎不能使用uncompyle6
